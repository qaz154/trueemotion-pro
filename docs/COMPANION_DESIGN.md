# TrueEmotion Companion — 详细设计文档

> 版本: v0.1 | 日期: 2026-05-12 | 基于 trueemotion-pro v1.16

---

## 1. 产品概述

**TrueEmotion Companion** 是一个 AI 情感陪伴 Web 应用，基于 trueemotion-pro 的 40+ 情感检测引擎，
提供有温度的对话体验和实时情绪可视化。

### 1.1 目标用户
- 18-35 岁中国年轻用户
- 需要情感出口但不想打扰朋友的人
- 日常压力大、想找个"树洞"的人
- 对 AI 陪伴感兴趣的技术尝鲜者

### 1.2 核心价值
- **懂你**：40+ 细粒度情感 + 复合情感（悲喜交加、吃醋等）+ 中文反讽识别
- **记住你**：关系递进系统，越聊越懂你
- **看得见**：实时情绪可视化，让情感变成看得见的画面

### 4.3 EmotionOrb 设计

EmotionOrb 是产品的视觉核心，基于 VAD 坐标实时变化：

| 属性 | 映射规则 | 示例 |
|------|---------|------|
| **颜色** | Valence → 色相 (负=红/橙, 正=蓝/绿) | sadness: 冷蓝, joy: 暖黄 |
| **饱和度** | Arousal (高唤醒=鲜艳, 低唤醒=灰调) | anger: 鲜红, melancholy: 灰蓝 |
| **大小** | Intensity (0-1 → 24px-80px) | 强烈情感=大球 |
| **脉搏速度** | Arousal (高=快, 低=慢) | anxiety: 快速脉搏 |
| **发光强度** | Confidence (高置信=强光晕) | 不确定时暗淡 |

**复合情感**：当检测到复合情感（如 bittersweet = joy + sadness），
Orb 显示渐变色（暖黄 → 冷蓝），两种颜色在球体表面流动。

### 4.4 EmotionBubbles 设计

背景浮动气泡，每个气泡代表一个检测到的情感：

- **气泡数量**：当前 `all_emotions` 中置信度 > 0.05 的情感
- **气泡大小**：置信度 × 最大半径
- **气泡颜色**：与 EmotionOrb 相同的色彩映射
- **气泡位置**：高置信度气泡向中心聚拢，低置信度在边缘
- **气泡动画**：缓慢浮动，新消息时气泡重新排列

## 5. 数据流

### 5.1 对话流程

```
用户输入文字
     │
     ▼
前端 WebSocket 发送 { text, user_id }
     │
     ▼
后端 WebSocket 接收
     │
     ▼
EmotionAnalyzer.analyze(text, user_id)
     │
     ├─→ 规则引擎检测情感
     ├─→ LLM 深度理解 (如果可用)
     ├─→ 反讽检测
     ├─→ 上下文分析
     ├─→ 生成共情回复
     └─→ 更新用户记忆
     │
     ▼
序列化 AnalysisResult → JSON
     │
     ▼
WebSocket 返回给前端
     │
     ├─→ 渲染 AI 回复气泡
     ├─→ 更新 EmotionOrb (颜色/大小/脉搏)
     ├─→ 更新 EmotionBubbles (背景气泡)
     └─→ 更新用户画像侧边栏
```

### 5.2 情绪记忆流程

```
每次对话结束
     │
     ▼
MemoryRepository.learn_pattern()
     │
     ├─→ 提取关键词
     ├─→ 记录情感 + 强度 + 时间戳
     ├─→ 更新 emotional_history
     └─→ 计算 relationship_level
     │
     ▼
存储到 memory/users/{user_id}.json
     │
     ▼
下次对话时加载历史
     │
     ▼
LLM 使用历史上下文生成更个性化的回复
```

## 6. 色彩映射系统

### 6.1 情感-色彩映射表

基于 VAD 坐标计算 HSL 色值：

| 情感 | VAD | 色相(H) | 饱和度(S) | 亮度(L) | 预览色 |
|------|-----|---------|-----------|---------|--------|
| joy | (0.85, 0.50, 0.70) | 45° | 85% | 65% | 🟡 暖金 |
| sadness | (-0.85, -0.30, -0.50) | 220° | 60% | 45% | 🔵 冷蓝 |
| anger | (-0.85, 0.70, 0.50) | 0° | 80% | 50% | 🔴 鲜红 |
| fear | (-0.65, 0.60, -0.40) | 270° | 50% | 40% | 🟣 暗紫 |
| anxiety | (-0.48, 0.58, -0.25) | 30° | 65% | 55% | 🟠 焦橙 |
| surprise | (0.30, 0.80, 0.30) | 60° | 90% | 60% | 🟡 亮黄 |
| love | (0.93, 0.35, 0.82) | 340° | 75% | 65% | 🩷 粉红 |
| trust | (0.65, 0.30, 0.70) | 150° | 50% | 55% | 🟢 森绿 |
| despair | (-0.92, 0.25, -0.72) | 240° | 40% | 30% | 🔵 深靛 |
| disgust | (-0.75, -0.10, -0.40) | 100° | 45% | 35% | 🟢 暗绿 |
| neutral | (0.00, 0.00, 0.00) | 0° | 0% | 60% | ⚪ 灰白 |

### 6.2 色彩计算公式

```typescript
function vadToColor(vad: [number, number, number]): string {
  const [v, a, d] = vad;
  // 色相: Valence 映射到色轮 (负=红/紫, 正=蓝/绿)
  const hue = ((v + 1) / 2) * 240; // -1→0°(红), 0→120°(绿), 1→240°(蓝)
  // 饱和度: Arousal 控制鲜艳程度
  const saturation = Math.abs(a) * 80 + 20; // 20%-100%
  // 亮度: Dominance 控制明暗
  const lightness = ((d + 1) / 2) * 40 + 30; // 30%-70%
  return `hsl(${Math.round(hue)}, ${Math.round(saturation)}%, ${Math.round(lightness)}%)`;
}
```
